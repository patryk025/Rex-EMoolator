# INE format — Inertia physics

!!! warning "Work in progress"
    The `.INE` format is only just being analysed — at this point there is no confirmed specification of its structure. This page is a stub and will be filled in as reverse-engineering progresses.

The `.INE` file stores data for the built-in 2D physics engine **Inertia**, used in *Reksio i Kretes w Akcji* ([`INERTIA`](../reference/INERTIA.md)).

## What is known so far

- The format is binary.
- It is associated with [`INERTIA`](../reference/INERTIA.md) objects.
