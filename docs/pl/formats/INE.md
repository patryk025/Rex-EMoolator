# Format INE — fizyka Inertia

!!! warning "W opracowaniu"
    Format `.INE` jest dopiero analizowany — na ten moment nie ma potwierdzonej specyfikacji jego budowy. Ta strona jest zalążkiem i zostanie uzupełniona w miarę postępu reverse-engineeringu.

Plik `.INE` przechowuje dane dla wbudowanego silnika fizyki 2D **Inertia**, użytego w grze *Reksio i Kretes w Akcji* ([`INERTIA`](../reference/INERTIA.md)).

## Co już wiadomo

- Format jest binarny.
- Powiązany jest z obiektami typu [`INERTIA`](../reference/INERTIA.md).
